Release Notes

Version 1.5.8 (1131)

  • New Installation and Upgrade Wizard
    Does not require a drive mapping to SYS:
    MailFilter can be installed on another volume as SYS:
    MailFilter can be installed in a seperate directory (useful for Novell Cluster Services installations)
  • MFConfig has been merged into MailFilter.
    Online Configuration: Press F4 in MailFilter main screen
    Offline Configuration: load mailflt -t config
  • "Legacy" MailFilter has been renamed to MFLT50.NLM. Only the LibC Version is named MAILFLT.NLM now.
  • MFNRM has been merged into MailFilter.
    If you have MailFilter running in the OS address space, you can enable MFNRM from the Configuration Editor.
    If you have MailFilter running in protected mode, you must load mailflt -t nrm to enable MFNRM.
  • Graphical Filter Editor for Windows included.
  • Updated MFFilterICE to match new configuration format.
  • Some memory corruption fixes in MailFilter.

Version 1.5.5 (1123)

  • Added MailCopy module - this module requires an approaite license.
  • Added paging to MFNRM.NLM
  • Created new FilterICE - can read/write CSV, TAB and binary filter files.
  • PFA mode now check the second Received header, as the first one will always be "PFAgent"
  • Bugfix: Virus Scanning module would send mails always to all parties.
  • Removed Filter2CSV

Version 1.5.4 (1119)

  • Spam DNS Blacklist feature is now a normal filter rule. Your old setting does not get converted automatically.
  • Daily Admin Message now includes log file.
  • Filter2CSV: Updated.
  • MailFilter Core: Fixed a bug where MFN files were left in the MFSCAN directory tree.
  • MailFilter Core: Fixed an abend in the attachment decoder.
  • MailFilter Core: Internal Configuration System rewrite.
  • Local Installer: Updated, should cause less problems.
