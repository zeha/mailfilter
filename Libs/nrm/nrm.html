1.0 What's New

The NetWare Remote Manager allows you to interact with a remote client using HTTP protocol. These HTTP server services allow you to receive a request and respond to it. This API can be used to extend the NetWare Remote Manager to include custom management and diagnostic information.

The October 2003 release added HTTP client services. You can now use the HTTP protocol to send a request and receive a reply to that request. These client services are only available on NetWare 6.5.
