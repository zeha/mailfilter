Rule Export

This allows you to export rules into either a binary MailFilter/ax Rule Package (as distributed with MailFilter/ax) or into a CSV file, which then can be edited using Excel or any other CSV compliant application.

The MailFilter/ax Rule package is compatible with the Rule package that can be exported using MFConfig.NLM.
The CSV file is compatible with the CSV produced by MFFilter2CSV.
